# Graph_OOP
Построение графиков функций с применением ООП
Visual studio 2019 C++
Консольное приложение, реализующее построение математический функций:
1) y = x
2) y = x^2
3) y = |x|
4) y = sin(x)
Построение происходит на заданном с клавиатуры отрезке.

Описание файлов:
Практика 2 курс.cpp - основной код программы
Grapg.h - описание классов и прототипы функций
Grapg.cpp - реализация методов классов
